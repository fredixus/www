Webpage description - project requirements

Welcome! On this webpage you can search, browse and comment your's favourite books. About 5000 book in our database. Project contains search feature, you can search book with ISBN, Title or Authors Name.

If you are intrested in details feel free to read description of filles enclosen bellow:

  • application.py - root file, handle with backend of web page. In application I set up end point for:
    1. index - main webside. Include description of another pages. Include a short writeup describing your project, whatâ€™s contained in each file.
    2. search - webpage with form including: 3 inputs, 1 button. Inputs for: ISBN, Name of Author, Title. You can type any of three and you will find the resoults. If you type not full sentence search works as well. Use the Submit button to search book in database.
    3. login - use your login and password to login in to your account. After login you can add and read comments.
    4. register - before you login, in first visit, you can create free account. Account gives you opportunity to rating books, writing revies, reading reviews of other users. Page load after registation.
    5. logout - you can use it only if you are login.
    6. notes - quick notes
    7. afterSearch - page like in google search, gives you results of your search.
  • style.css - file with CSS stylesheet. Format WWW web page, add some colors, border, paddings and margins, format text. I use Boostrap CSS framework to handle with styles. It is more quickly that creating new full stylesheet.
